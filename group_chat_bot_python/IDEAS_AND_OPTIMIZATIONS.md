# Идеи и план по максимальной оптимизации NeuroNika

Этот файл создан по запросу пользователя, чтобы сохранить все идеи по улучшению, полировке и оптимизации бота.

## 1. Архитектурная оптимизация (Рефакторинг)
**Проблема:** Файл `commands.py` содержит более 3300 строк кода. Это делает его сложным для поддержки, изменения и поиска ошибок.
**Идея:** Разбить команды на роутеры (Aiogram 3 Routers) по категориям:
- `app/bot/routers/economy.py` — команды магазина, перевода печенек, налогов, долгов.
- `app/bot/routers/games.py` — казино, дуэли, башня, монетка, рыбалка.
- `app/bot/routers/profile.py` — `/me`, `/bio`, `/mybirthday`, настройки профиля.
- `app/bot/routers/admin.py` — модерация, `/cookie_rain`, `/jail`.
- `app/bot/routers/ai_features.py` — генерация картинок, сигны, работа с памятью.
- `app/bot/routers/general.py` — `/help`, статистика, топы.

## 2. Апгрейд Экономики и Игр
- **Рейтинг дуэлей:** Сохранять статистику побед и поражений в дуэлях (Камень-Ножницы-Бумага и кости). Добавить отдельный `/top duels` и показывать винрейт в `/me`.
- **Ежедневный турнир:** Проводить раз в день (например, вечером). Участие стоит Х печенек. Все взносы формируют призовой фонд. Победитель определяется случайным образом или серией мини-игр среди участников, победитель забирает банк.
- **Налоги и инфляция:** Для баланса крупных сумм добавить комиссию на перевод (`/give`) для сумм выше 10,000 печенек.
- **Умные лимиты:** Ограничение на количество сыгранных игр в казино в час, чтобы люди не фармили (или не сливали) все печеньки моментально.

## 3. Максимальная прокачка общения (ИИ)
- **Кнопка "Не запоминать":** Под сообщениями ИИ, если он сообщает "Я запомнила...", добавить инлайн-кнопку "❌ Забудь", чтобы пользователь мог сразу удалить неправильно понятый факт.
- **Команда `/forget <сумма/факт>`:** Дать пользователям возможность вручную удалять факты о себе из памяти бота.
- **Улучшение контекста:** Сейчас история чата склеивается. Нужно оптимизировать сборку контекста, чтобы бот лучше понимал сарказм и шутки (добавить в системный промпт инструкции по сарказму).
- **Разделение моделей:** Если генерация картинок занимает долгое время, отправлять пользователю промежуточное "⏳ Рисую..." (typing action), чтобы он понимал, что бот работает.

## 4. Настройки чата для Администраторов
- Добавить команду `/chatsettings`, которая открывает меню (InlineKeyboardMarkup).
- В меню можно включать/выключать модули для конкретного чата:
  - Авто-дропы (да/нет)
  - Викторины (да/нет)
  - Казино и азартные игры (да/нет)
  - ИИ картинки (да/нет)

## 5. Оптимизация производительности и кода
- **Кэширование:** Кэшировать настройки чатов (ChatSettings) в памяти на 5-10 минут, чтобы не дергать базу данных `Supabase` при каждом сообщении.
- **Оптимизация базы данных:** Добавить индексы в Supabase для таблиц с логами, чтобы команда `/top` работала мгновенно даже при миллионах записей.
- **Обработка исключений:** Обернуть вызовы к внешним API (OpenAI / Polza) в retry-декораторы с экспоненциальной задержкой, чтобы избежать крашей при сетевых ошибках.
