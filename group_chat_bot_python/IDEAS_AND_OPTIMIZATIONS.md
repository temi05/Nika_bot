# Глобальный План Улучшения и Оптимизации NeuroNika (Ultimate Upgrade)

Этот документ содержит утвержденный план по максимальной прокачке бота.
*Учтено: Бот работает на Render (оптимизация ресурсов).*

## 1. Инфраструктура и 100% Надежность (Фундамент)
- **Продвинутое кэширование:** Использование `asyncio` in-memory кэша (с TTL) для `get_chat_settings` и `get_user`. Снижает задержки и нагрузку на Supabase.
- **Фоновые задачи (Background Tasks):** Генерация картинок и сигн выносится в фоновые процессы. Бот сразу отвечает "⏳ Рисую..." и остается свободным для других команд.
- **Железобетонный Retry:** Умный механизм повторных попыток для API OpenAI/Polza. Бот устойчив к сетевым сбоям.

## 2. Умный Ассистент (Прокачка Мозгов)
- **Выполнение команд из текста:** ИИ сможет выполнять транзакции и команды через обычное общение (Function Calling).
- **Оптимизация контекста:** Умное управление историей сообщений для экономии токенов и ускорения ответов.
- **Кнопка "Забыть":** Inline-кнопка под сообщениями о сохранении памяти для мгновенного удаления ложных фактов.

## 3. Социальность и Соревнования (Экономика 2.0)
- **Система Достижений (Ачивки):** Бейджи в профиле (`/me`) за активность и игровые успехи.
- **Ежедневные Турниры:** Автоматический розыгрыш банка среди участников раз в сутки.
- **Глобальные оповещения:** Крутые уведомления о джекпотах и турнирах на весь чат.
