# Статус разработки NeuroNika

## Описание
Этот файл содержит список выполненных задач и текущий прогресс разработки бота.

## Выполненные задачи
- [x] Создан и подключен глобальный обработчик ошибок (`app/bot/routers/error.py`).
- [x] Восстановлена целостность `games.py` после повреждения (исправлены `/scratch` и `/mine`).
- [x] Исправлен `NameError` в `general_admin.py` (вызов `run_mine`).
- [x] Исправлена ошибка `Bad Request: can't parse entities` при вызове команды `/help`.
    - Причина: Использование символов `<` и `>` в HTML-сообщениях.
    - Решение: Символы заменены на `&lt;` и `&gt;`, а также добавлен `parse_mode="HTML"` для корректного отображения в файлах:
        - `app/bot/commands.py`
        - `app/bot/rp_commands.py`
        - `app/bot/feedback.py`
- [x] Проверены все остальные команды на наличие подобных ошибок.
- [x] Исправлена ошибка `[AI:cookie_gift_error]` (BAD_REQUEST) при вызове генерации сообщения для передачи печенек: добавлены роли `user` и `system`, так как некоторые провайдеры (Together AI) отклоняют запросы только с `system`.
- [x] Исправлено "молчание" бота (когда сообщения тихо отклонялись с BAD_REQUEST). Теперь вся история собирается в один блок `user` (ранее история отправлялась отдельными сообщениями `user` подряд, что запрещено у некоторых LLM).
- [x] Добавлено логирование скрытых ошибок API в функциях `_retry_empty_reply`.
- [x] Добавлена новая команда `/casino <сумма>` для игры на печеньки (рулетка) с разными шансами (проигрыш, x1.5, x2, x5). Команда добавлена в меню `/help`.

- [x] Глобальное отключение всех мини-игр по просьбе Ники для борьбы со спамом и инфляцией.
- [x] Налог на переводы печенек поднят до 15% для предотвращения RMT.
- [x] В магазин (`/shop`) добавлено строгое предупреждение о запрете покупки печенья за реальные деньги.
- [x] Лимит теоретических ставок снижен до 15 000 печенек.
- [x] Введена выдача 1-3 печенек за каждое сообщение в чате (раз в 60 сек) как основной источник дохода.
