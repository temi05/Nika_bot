# План улучшения Telegram-бота Nika

Этот файл содержит список идей и задач для улучшения бота.

## ✅ ВЫПОЛНЕНО

### ИИ-функционал (v2.6 — "Личность освобождена")
- [x] **Полная смена личности**: Удалены все упоминания ИИ, кода и "программной природы". Ника теперь — человек, альтер-эго.
- [x] **Разговорный стиль**: Внедрены слова-паразиты, сленг и человеческие интонации.
- [x] **Интеллектуальный поиск**: Исправлен поиск юзеров (Ника/Ники/Нику — теперь находит всех по корню слова).
- [x] **Опросы (create_poll)**: Добавлен инструмент создания нативных опросов Телеграма (анонимные, множественный выбор).
- [x] **Напоминания (set_reminder)**: Добавлен инструмент напоминаний через setTimeout с тегом пользователя.
- [x] **Контекст ответов (Replies)**: Ника видит, на какое сообщение ей отвечают.
- [x] **Распознавание стикеров**: Видит смайлики и типы стикеров (видео/анимированные).
- [x] **Безопасная отправка (safeSendMessage)**: Защита от ошибок HTML-разметки (например, <3).

### Ранее выполненное
- [x] **Дневник чата (Memory)**: Суммаризация последних 12 сообщений в постоянную память Supabase.
- [x] **Досье юзеров**: Система заметок (notes) и био (bio).
- [x] **Модерация**: Варны и муты (Ника сама выбирает время мута от 15 мин до 24 ч).
- [x] **Система XP/Level**: Прокачка в чате и ежедневный бонус /daily.

### Новые достижения (v2.7 — "Ассоциативный Разум")
- [x] **Память 2.0 (GraphRAG-lite)**: Внедрен двухэтапный поиск ассоциаций. Ника теперь понимает связи (кто чей друг, кто что купил) и строит логические цепочки.
- [x] **Умное извлечение связей**: ИИ выделяет Субъекты, Связи и Объекты для хранения в графе знаний.
- [x] **Исправление Идентичности**: Бот корректно распознает анонимных админов как одну личность и запоминает выбранные пользователем имена (`set_user_name`).
- [x] **Личное Досье**: Прямая инъекция `ai_notes` в промпт для 100% надежного запоминания важных фактов о юзере.
- [x] **Стабильность логов**: Исправлены ошибки `ReferenceError` и улучшена отладка БД.
