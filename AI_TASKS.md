# План улучшения Telegram-бота Nika

Этот файл содержит список идей и задач для улучшения бота, чтобы контекст сохранялся между сессиями.

## ✅ ВЫПОЛНЕНО

### ИИ-функционал (v2 — "Умная НейроНика")
- [x] **10 Function Calling инструментов**: bio, notes, stats, profile, birthdays, warn, mute, poll, reminder
- [x] **Пассивное наблюдение**: буфер последних 15 сообщений чата (RAM) для контекста ИИ
- [x] **Анализ фото**: Vision API — ИИ может комментировать картинки и мемы
- [x] **Контекст времени**: ИИ знает текущее время и день недели
- [x] **Расширенный промпт**: ИИ знает все команды бота, умеет подсказывать, модерировать по запросу админа
- [x] **Напоминания**: setTimeout-напоминания через ИИ ("напомни через 30 мин")
- [x] **Опросы**: ИИ может создавать голосования по запросу
- [x] **Модерация через ИИ**: варны и муты (ТОЛЬКО по запросу админа или при явных нарушениях)
- [x] **Единый обработчик tool calls** (executeToolCall) вместо inline if/else

### Предыдущие улучшения
