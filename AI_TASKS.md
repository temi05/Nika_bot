# План улучшения Telegram-бота Nika

Этот файл содержит список идей и задач для улучшения бота, чтобы контекст сохранялся между сессиями.

## 🛠 Технические улучшения
- [ ] **Оптимизация фильтрации мата**: Предварительная компиляция регулярных выражений вместо создания новых на каждое сообщение.
- [ ] **Безопасность Webhook**: Добавить проверку `X-Telegram-Bot-Api-Secret-Token` для защиты эндпоинта.
- [ ] **Логирование**: Внедрить нормальную систему логирования (например, `winston` или просто более структурированные логи в файл/Supabase).
- [ ] **Кэширование**: Улучшить кэширование пользователей, чтобы `handleReaction` не делал лишних запросов в БД.
- [ ] **Централизация логики XP**: Вынести расчет уровней в `database.js` полностью, чтобы избежать дублирования формул в `commandHandler` и `messageHandler`.
- [ ] **Graceful Shutdown**: Корректное завершение работы сервера при обновлении.

## ✨ Улучшение UX/UI (Premium Feel)
- [ ] **Форматирование ответов**: Сделать все сообщения более красивыми, используя HTML-теги, отступы и тематические эмодзи.
- [ ] **Профили с фото**: Добавить отображение аватарки пользователя в `/me` (через ссылку или как-то иначе).
- [ ] **Интерактивность**: Добавить больше inline-кнопок для навигации.
- [ ] **Лоадеры**: Показывать временные сообщения "Загружаю..." для тяжелых команд (например, `/top`).

## 🚀 Новые функции
- [ ] **Глобальные настройки чата**: Возможность админам включать/выключать модули (XP, репутация, фильтр мата).
- [ ] **Ежедневные бонусы**: Команда `/daily` для получения печенек раз в день.
- [ ] **Улучшенная капча**: Более сложные или разнообразные проверки для защиты от умных ботов.
